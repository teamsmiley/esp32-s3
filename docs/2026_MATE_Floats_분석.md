# 2026 MATE Floats! 미션 분석

> **얼음 아래에서 (Under the Ice)** | 총 70점

---

## 미션 개요

2026년 MATE ROV 대회의 "얼음 아래 플로트" 미션은 NSF의 **GO-BGC 프로젝트**에서 영감을 받았습니다.  
GO-BGC는 전 세계 500개의 해양 모니터링 플로트를 배치하여 해양 순환·화학·생물·건강을 모니터링하는 프로젝트입니다.

- 지역 대회: 수면에 얼음이 있다고 가정한 시뮬레이션 환경
- **세계 챔피언십**: 캐나다 국립연구원(NRC) 실제 얼음 탱크에서 진행
  - 탱크 규모: 90m × 12m × 3m
  - 온도: 최저 -25°C
  - 얼음 생성 속도: 2.5mm/시간

## 단계별 상세 요건

### ① 플로트 설계 및 제작 (5점)

- 대회 전 DOC-004(비-ROV 장치 설계 문서) 제출 필수
- 지역 대회는 제출 여부를 주최 측에 확인 (미제출 시 0점)
- 플로트를 제품 시연 스테이션까지 운반해야 점수 인정

### ② 배포 전 통신 (5점)

플로트를 물에 투입한 직후, 지상 수신기로 **정의된 데이터 패킷**을 최소 1회 전송해야 합니다.

**데이터 패킷 필수 포함 항목:**

- 회사 번호 (Company Number)
- 시간 데이터 (UTC / 로컬 / 플로트 시작 후 경과 시간)
- 압력 데이터 (Pa 또는 kPa) 및/또는 수심 데이터 (m 또는 cm)

**예시:**

```
EX01  1:51:42 UTC  9.8 kPa  1.00 meters
```

> 참고: 첫 번째 프로파일 시작 전 전송해야 하며, 먼저 하강하면 통신 점수 미부여

### ③④ 수직 프로파일 (각 25점, 총 50점)

**수직 프로파일 정의:**  
플로트가 수면에서 출발 → 수심 2.5m 하강 유지 → 수심 40cm 상승 유지

> **프로파일 횟수**: 공식 PDF는 정확히 **2회**(1회차/2회차)를 점수 인정합니다.  
> 추가 프로파일 시도는 보정 목적(아래 "추가 프로파일 허용" 절)으로만 허용되며 별도 점수는 없습니다.

#### 수심 2.5m 유지 조건

- 플로트 **하단부** 기준으로 2.5m (±33cm)
- 허용 범위: **2.27m ~ 2.83m** (PDF 명시 문구; 아래 경고 참고)
- 30초 동안 유지 (범위 이탈 시 처음부터 재측정)
- 데이터 패킷 **7개 연속** (0, 5, 10, 15, 20, 25, 30초) 필요

> ⚠ **PDF 내부 모순**: 공식 PDF가 명시한 범위는 `2.27 ~ 2.83`이지만, 같은 PDF의 센서 오프셋 예시 계산을 역산하면 실제 의도는 `2.17 ~ 2.83`(±33cm 대칭)으로 보입니다.  
> — 예시(line 202-204): "센서가 bottom보다 25cm 위에 있으면 센서 범위는 1.92 ~ 2.58m" → bottom 환산 시 `1.92+0.25 = 2.17` ~ `2.58+0.25 = 2.83`  
> 코드는 보수적으로 `2.27 ~ 2.83`을 사용. 시합 전 채점관에게 어떤 해석을 쓸지 확인하면 점수 손실 위험 감소.

#### 수심 40cm 유지 조건

- 플로트 **상단부** 기준으로 40cm (±33cm)
- 허용 범위: **7cm ~ 73cm** (대칭, 이슈 없음)
- 30초 동안 유지 (범위 이탈 시 처음부터 재측정)
- 데이터 패킷 **7개 연속** 필요

#### 센서 오프셋 통보 의무 (중요)

PDF line 201-205 / 220-223 — 센서가 플로트 bottom 또는 top에 정확히 위치하지 않을 경우, **시합 전 채점관(judge)에게 다음을 통보할 의무**가 있습니다:

1. 센서의 정확한 위치 (bottom 또는 top으로부터의 거리)
2. **조정된 센서 범위** (오프셋 반영)

PDF 예시: 센서가 bottom보다 25cm 위에 있으면, bottom이 2.5m일 때 센서는 2.25m. 따라서 센서 기준 허용 범위는 `1.92m ~ 2.58m` (PDF 본문 기준; 위 경고 참고).

**우리 플로트 적용**: 플로트 총 높이 **13 in (33.02cm)** = 12 in 튜브 + 0.5 in × 2 캡. 센서는 **bottom cap 안에 플로트 바닥면과 일치**(flush)되어 있어 오프셋 0. 따라서 센서값 = bottom 깊이:

| 미션 목표 | bottom 깊이 | 센서 깊이 (= bottom) |
|---|---|---|
| 2.5m hold | 2.27 ~ 2.83 m | 2.27 ~ 2.83 m (동일) |
| 40cm hold | top 0.07 ~ 0.73 m → bottom 0.4002 ~ 1.0602 m | 0.4002 ~ 1.0602 m |

`SHALLOW_*` 밴드값은 코드에서 `0.07 + FLOAT_HEIGHT_M`, `0.73 + FLOAT_HEIGHT_M`으로 자동 계산되므로 13인치 적용 시 위 값으로 갱신됩니다.

> 시합 등록·배포 전 위 정보를 채점관에게 사전 통보 필요.  
> 패킷 출력은 `reportedDepth()`(bottom 환산)로 통일되어 있으므로 채점관에게는 "패킷의 meters 값은 플로트 BOTTOM 기준"이라고 설명.

#### 추가 프로파일 허용

- 패널티 발생이나 수심 미달이 의심될 경우 추가 프로파일 시도 가능
- 더 높은 점수의 프로파일로 대체 가능 (단, 프로파일 내 항목 혼용 불가)
- 단, 점수 인정은 여전히 **최대 2회분만**

### ⑤ 회수 후 데이터 전송 (10점)

- 플로트 회수 후 수신기로 데이터 패킷 무선 전송
- **전체 전송 (10점)**: 두 프로파일 전체에서 5초마다 1개 이상의 패킷
- **최소 전송 (5점)**: 프로파일 중 최소 1개 이상의 패킷 (하강 이후 데이터만 인정)

### ⑥ 수심-시간 그래프 (10점)

- X축: 시간 / Y축: 수심
- 최소 **20개 데이터 포인트** 필요
- **컴퓨터 사용 필수** (손으로 그린 그래프 불인정)
- 20개 미만이거나 플로트 통신 실패 시 → MATE 제공 데이터 사용 가능

## 플로트 규격 요건

### 물리적 규격

| 항목          | 규격                  |
| ------------- | --------------------- |
| 전체 높이     | 1m 미만 (안테나 포함) |
| 지름/폭       | 18cm 미만             |
| 테더/에어라인 | 불가 (완전 독립 운용) |
| 분리형 구조   | 불가                  |

### 회수 장치 (2026 신규 요건 ★)

- 플로트에 **MATE ROV가 잡을 수 있는 구조물** 필수 장착
- 최소 **5cm 폭**, 최소 **5cm 돌출**
- 예시: U-볼트 (#310 이상), 로프/와이어 루프 등
- 18cm 최대 지름 초과 가능 (회수 장치에 한해 예외)
- 여분의 장치 설치 권장 (빠른 회수 = 데이터 분석 시간 확보)

### 부력 엔진

- **부력 엔진 정의**: 플로트 내부 유체를 외부로 이동시켜 밀도 변경
- 인정되는 방식: 모터로 공기 또는 액체 이동
- **불인정 방식**: 프로펠러 또는 워터젯 추진 (직접 추력 방식)

## 전기/배터리 규정

### 배터리 종류 (2026 변경 사항 ★)

| 배터리 종류           | 최대 퓨즈 크기 |
| --------------------- | -------------- |
| NiMH AA               | 2.0A           |
| NiMH C 또는 D         | 5.0A           |
| NiMH 9V               | 200mA          |
| NiMH / AGM 12V (대형) | 5.0A           |

- ✅ 허용: NiMH, AGM
- ❌ 불허: 알카라인, LiPo, 12V 야외용 충전 배터리
- 최대 전압: 12VDC / 최대 전류: 5A

### 퓨즈 규정

- **단일 퓨즈** 사용 필수 (모든 전원 통합)
- 배터리 양극 단자에서 **5cm 이내** 설치
- 퓨즈 종류: ATO 블레이드, MINI 블레이드, 또는 카트리지 (색상 코드 준수)
- 최소 32VDC 정격 필수
- 투명 하우징 또는 외함 제거 시 육안 확인 가능해야 함

### 압력 해방 장치 (ELEC-NRD-006)

하우징 내부 압력이 외부보다 높을 경우 열리는 구조 필수:

1. **압력 해방 구멍**: 최소 2.5cm 직경 (고무 마개는 마찰식만 허용)
2. **팝오프 엔드캡**: O링 밀봉, 최소 2.5cm 직경

> ❌ 압력 해방 밸브 불가 / ❌ 호스 클램프·Twist-Tite 방식 불가

## 세계 챔피언십 특이사항

- **얼음 두께**: 1cm ~ 5cm
- **투입 구멍**: 1m × 1m
- **수조 용액**: EGADS 용액 (비중 약 1.025)
- **플로트 회수**: MATE 직원이 별도 ROV로 회수 (회수 중 대회 시간 정지)
- **데이터 전송**: 데이터 스테이션까지 이동 시, 도착 시점에 시간 재개
- **방한 장비** 지참 권장 (특히 방수 장갑)
