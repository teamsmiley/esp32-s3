## 무선 명령 (ESP-NOW 연결 후)

지상국 → 플로트 명령 프로토콜에 다음을 포함:

- [ ] `CMD_ZERO` — 깊이 0점 재보정 트리거
  - 배경: BOOT 버튼은 물속에서 못 누름. 배포 직후 수면에서 마지막 보정용으로 무선 트리거 필요
  - 동작: 플로트가 명령 받으면 `calibrateZero()` 실행하고 새 offset 회신
- [ ] `CMD_PING` — 연결 확인 (지상국이 응답 시간 측정)
- [ ] `CMD_DUMP_LOG` — 회수 후 로컬 로그 전체 송신 트리거
- [ ] `CMD_START_MISSION` — 자율 프로파일 시퀀스 시작
